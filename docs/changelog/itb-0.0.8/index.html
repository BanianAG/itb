ITB 0.0.8

April 10, 2026

This release of ITB 0.0.8 focuses on practical workflow improvements, broader customization options, and measurable performance gains for data modeling.

📦 Release Overview​

This release bundles significant improvements across components:

ComponentVersionFocus
WebApp0.0.87 UI/UX features + Performance optimization
API0.0.8No changes (only alignment with WebApp)

🎯 WebApp 0.0.8​

7 UI/UX Features + Comprehensive Performance Optimization

Highlights​

🎯 Advanced Workflows​

  • ✨ Drag Entity from Sidebar to Canvas - Intuitive drag-and-drop entity placement
  • ✨ Drag & Context Menu for Related Entities - Explore relationships without sidebar searching
  • ✨ Comprehensive Canvas Context Menus - Right-click actions for entities, edges, and canvas
  • ✨ User Settings Dialog - Centralized preferences for language, theme, canvas behavior, and grid
  • ✨ Generate All Artifacts Button - One-click artifact generation with progress tracking
  • ✨ Entity Tree Auto-Reveal - Canvas selection automatically reveals and scrolls to entity in sidebar

⚡ Performance​

  • ⚡ Canvas Rendering Performance Boost - 60%+ render time reduction for large viewpoints
    • Viewport culling: Only visible elements rendered
    • Shared data maps for entity types and attributes
    • Lazy edge labels (hidden below 50% zoom)
    • Singleton dialogs: One instance for entire canvas
    • Cached handle positions for edge routing
    • Debounced viewpoint switching batched via requestAnimationFrame
    • Result: Large 30-entity, 40-edge viewpoints now render in under 10 seconds (vs. ~25 seconds)

→ View WebApp 0.0.8 Release Details

🎓 Summary​

ITB 0.0.8 is a focused update that improves day-to-day modeling workflows and responsiveness. The 7 UI features reduce interaction overhead, and the 60%+ rendering improvement makes larger viewpoints easier to work with in regular usage.

Key Achievements:

  • ✅ Intuitive drag-and-drop workflows
  • ✅ Context-aware actions reduce clicks
  • ✅ Customizable workspace settings
  • ✅ 60%+ performance improvement
  • ✅ Large models now production-ready

⚠️ Upgrade Notes​

New Docker Compose File and new .env File! Please update.

Recommended Actions​

  1. Backup your current workspace: Export any critical models before upgrading
  2. Clear browser cache: Some CSS and layout changes may require cache refresh
  3. Test large models: If working with models exceeding 50 entities, test the new performance improvements

Compatibility​

  • ✅ Fully backward compatible with existing models
  • ✅ Storage layout unchanged
  • ✅ API format unchanged

🙏 Thank You​

Thank you to all contributors and users who shared feedback and tested pre-release versions. That input helped refine this release.
